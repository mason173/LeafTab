LeafTab

LeafTab 专注于一件事：把“打开新标签页”变成一个稳定、顺手、可控的高效起点。 你可以获得键盘优先的搜索体验、灵活的快捷方式管理，以及端到端加密备份、WebDAV 同步与自托管能力。项目完全开源且免费。

  • 本地优先：默认数据保存在浏览器本地
  • 强搜索：标签页 / 书签 / 历史 / 搜索引擎统一入口
  • 可同步：官方后端、自定义后端、WebDAV 多种方案
  • 端到端加密备份：备份数据可加密后再上传
  • 可扩展：图标库、壁纸、天气组件与多布局模式
  • 完全开源免费：代码公开、社区可参与、可自行构建

为什么选择 LeafTab

我们不是把新标签页做成“复杂操作系统”，而是把常用工作流打磨到足够顺滑。 它对新用户友好，也给重度用户留足了控制力。

Search

键盘优先搜索

支持命令前缀和多源检索，让信息查找更快，减少鼠标切换成本。

Sync

同步与备份可控

支持账号同步与 WebDAV 备份，数据迁移、导入导出更安心。

Self-host

后端可自托管

登录、同步、统计接口可切换到自定义服务器，适配不同部署需求。

E2EE

端到端加密备份

在支持的备份流程中，数据可先加密再同步，降低明文暴露风险。

WebDAV

WebDAV 同步支持

你可以接入自己的 WebDAV 服务，实现跨环境的数据备份与恢复。

Open Source

完全开源免费

仓库公开、可审计、可二次部署，社区用户可以直接参与共建。

核心特性

围绕效率、可控与隐私构建，避免不必要的权限和噪音设计。
