新标签页不该只是一个空白页 让搜索、启动、同步都在一个地方完成 从个人使用到自托管都能平滑过渡

不是堆功能，而是把常用能力排得更顺。

官网这次按 `DESIGN.md` 重新收敛到更克制的黑白体系，只留必要的层级、信息密度和一处产品绿色，让产品能力自己说话。

Search

统一搜索入口

搜索引擎、站内搜索、书签、历史记录、当前标签页切换，都可以从一个输入框快速完成。

Sync

同步方式自己选

可选云同步、WebDAV 备份或自托管后端，不强迫你把数据和账号绑定在单一服务里。

Privacy

默认本地优先

大多数偏好和交互默认只留在浏览器本地，联网功能按需开启，可选权限按需申请。

UI

氛围感但不喧宾夺主

场景模式、主题色、壁纸和布局节奏经过收敛，让新标签页既有状态也不影响进入工作。

Open

社区版可直接分发

社区版可以带着本地图标库一起打包发布，弱网或无法访问外部图标服务时也能正常显示。

Control

从个人习惯出发

快捷方式、搜索顺序、场景偏好和同步策略都可以围绕你的使用方式调整，而不是反过来适应产品。

快捷搜索 壁纸与主题 命令式交互

把隐私边界说清楚，比口号更重要。

LeafTab 的同步、Google 登录、WebDAV 和匿名统计都按“主动开启、最小化使用、可替换后端”来设计。这些边界，我们单独写成了隐私政策。

默认本地优先

大多数偏好设置和日常搜索交互默认只保留在浏览器本地。

联网能力按需启用

云同步、WebDAV、Google 登录与匿名统计都是可选能力，不会默认强塞给所有用户。

社区版也能稳定使用

社区版登录流程已统一为网页登录 fallback，不再被扩展 ID 变化绑死。
