Place the local Chinese bookmark embedding model files here before packaging:

- `config.json`
- `tokenizer.json`
- `tokenizer_config.json`
- `special_tokens_map.json`
- `vocab.txt`
- `onnx/model_quantized.onnx`

The repository also includes `scripts/vendor-bookmark-zh-model.sh` to download these assets when network access is available.
